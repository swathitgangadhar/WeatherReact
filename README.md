# Weather App

A dynamic weather application built with React and the OpenWeather API, providing users with real-time, location-based weather updates. Utilized React Hooks, conditional rendering, and the fetch API to efficiently manage data flow and enhance user experience.

Animated weather backgrounds — rain streaks, falling snowflakes, twinkling stars, and glowing sun haze, all driven by the live weather condition
Dynamic color themes per condition (stormy purple, snowy light blue, rainy slate, cloudy gray, sunny sky, night indigo)
5-Step Forecast row showing time, icon, temp & humidity
UV Index card with color-coded bar (green → yellow → orange → red → purple) pulled from the OneCall API
Wind card with gust speed and compass direction
Pressure card with visibility in km
Humidity bar, feels-like temp, and high/low range


## Setup & Run
npm start
